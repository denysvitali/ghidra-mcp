# GhidraMCP - Production Dependencies
# Install with: pip install -r requirements.txt
#
# As of v5.14.2 the MCP bridge is implemented in Go (bridge-go/).
# The bridge binary is `ghidra-mcp-bridge`; prebuilt releases are at
# https://github.com/xebyte/ghidra-mcp/releases. No Python runtime is
# needed to run the bridge.
#
# Python is still used by:
#   * the standalone debugger server (debugger/, see requirements-debugger.txt)
#   * fun-doc/ (see fun-doc/requirements.txt)
#   * integration tests + tooling (see requirements-test.txt)
